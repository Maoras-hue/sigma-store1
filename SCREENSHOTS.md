# Screenshots for Sigma Store

Take these screenshots before submitting to marketplace:

## Required Screenshots (10+)

1. **Homepage** - Full homepage with products
2. **Product Page** - Single product view
3. **Cart Page** - Shopping cart with items
4. **Checkout Page** - Guest checkout form
5. **Login/Signup** - Authentication page
6. **Admin Dashboard** - Admin panel view
7. **Seller Dashboard** - Seller view
8. **Mobile View** - Homepage on mobile
9. **Dark Mode** - Dark theme enabled
10. **Order Tracking** - Track order page

## Screenshot Tools

- **Windows:** Snipping Tool / Print Screen
- **Mac:** Cmd + Shift + 4
- **Browser DevTools:** Responsive mode

## Image Requirements

- Format: PNG or JPG
- Size: 1200x800px minimum
- File size: < 500KB each
